# project-3-bank-management-system
A simple  Python console-based bank management system that allows users to deposit, withdraw, and check their account balance.
🏦 Bank Management System

A simple Python console-based Bank Management System built using Python.

##Features
1. Create account
2. Deposit money
3. Withdraw money
4. Check account balance
5. Exit the program

##Concepts Used
1. Variables
2. Functions
3. Loops
4. Conditional Statements ("if-elif-else")
5. User Input
